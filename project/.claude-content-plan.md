## Pipeline per batch (established, keep identical)
1. Write each article to `scratchpad/articles/NN-slug.html`, 2000+ Thai words
   (estimate: Thai chars / 3.5 + Latin words ≥ 2000). Model structure EXACTLY
   on `project/learn/bookmark.html`: same `<style>` block, breadcrumb, `.tldr`,
   `.steps`, `<details>` FAQ, and a `@graph` with Organization + WebSite +
   BreadcrumbList + HowTo + TechArticle + FAQPage. Use `https://www.blogkub.com`
   canonical URLs. Include 3–5 natural internal links to other `/learn/` articles.
2. Render an AUTHENTIC screenshot per feature with `scratchpad/shots-bN.cjs`
   (uses `renderlib.cjs` `getCss(templateId, primary, accent, font, radius)` +
   the feature's real CSS from `builder-app.js`). Output ≥1200px wide to
   `scratchpad/featimgN/slug.png`. Local server: `python3 -m http.server 8899`
   from `project/`. Playwright at `/opt/node22/lib/node_modules/playwright`,
   chromium `/opt/pw-browsers/chromium`.
3. Publish with `scratchpad/publish-bN.cjs`: domain→www, inject canonical/og,
   add `mainEntityOfPage` + `image` ImageObject to TechArticle, insert Blogger
   `<figure class="separator">` (hosted URL, NOT base64) after `.tldr`, copy PNG
   to `project/learn/images/`.
4. Update `project/sitemap.xml` + `project/sitemap-images.xml` (append `<url>`).
5. Add the new cards to `project/learn/index.html` "คู่มือฟีเจอร์" grid.
6. Add 1–2 natural backlinks from existing articles → new ones.
7. Validate JSON-LD parses, figure present, canonical present. Commit + push.
