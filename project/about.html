เกี่ยวกับเรา

สร้างธีม Blogger ให้ ทุกคนทำเองได้ โดยไม่ต้องเขียนโค้ด

BlogKub คือเครื่องมือช่วยสร้างและตกแต่งธีม (template) สำหรับ Blogger ที่ทำงานในเบราว์เซอร์ทั้งหมด ใช้ฟรี ไม่ต้องสมัครสมาชิก และไม่ต้องมีพื้นฐานการเขียนโค้ด

BlogKub คืออะไร

Blogger เป็นแพลตฟอร์มทำบล็อกฟรีที่ทรงพลัง แต่การปรับแต่งธีมให้สวยและทำ SEO ให้ถูกต้องมักต้องยุ่งกับไฟล์ XML ที่ซับซ้อน ซึ่งเป็นกำแพงสำหรับคนทั่วไป BlogKub เกิดขึ้นเพื่อทลายกำแพงนั้น ด้วยการให้คุณประกอบหน้าเว็บจาก “บล็อก” ที่ลากวางได้ ปรับข้อความ สี ฟอนต์ และเปิด/ปิดฟีเจอร์ต่าง ๆ แล้วกดดาวน์โหลดเป็นไฟล์ Blogger XML ที่พร้อมอัปโหลดใช้งานจริง

สิ่งที่เราเชื่อ

ฟรีและเข้าถึงง่าย

เครื่องมือที่ดีไม่ควรมีราคาเป็นกำแพง ทุกคนควรทำเว็บสวย ๆ ได้

ความเป็นส่วนตัวมาก่อน

ทำงานฝั่งผู้ใช้ 100% ข้อมูลโปรเจกต์อยู่ในเบราว์เซอร์ของคุณ ไม่ถูกส่งขึ้นเซิร์ฟเวอร์

คุณภาพระดับมืออาชีพ

ธีมมาพร้อมโครงสร้าง SEO, Schema และความเร็วที่ถูกหลักตั้งแต่แกะกล่อง

ทำงานอย่างไร

ทุกอย่างเกิดขึ้นในเบราว์เซอร์ของคุณเอง เราไม่มีการอัปโหลดข้อมูลขึ้นเซิร์ฟเวอร์ ขั้นตอนใช้งานง่ายเพียงไม่กี่ขั้น

  • เลือกเทมเพลต ที่ตรงกับแนวเว็บของคุณ ระบบติดตั้งบล็อกและฟีเจอร์ที่จำเป็นให้อัตโนมัติ
  • ปรับแต่ง ข้อความ รูป สี ฟอนต์ และจัดลำดับบล็อกด้วยการลากวาง
  • ดาวน์โหลด XML แล้วนำไปอัปโหลดเข้า Blogger ผ่านเมนู ธีม → กู้คืน/สำรอง

อยากเห็นว่าแต่ละฟีเจอร์ทำงานอย่างไร ดูคู่มือละเอียดได้ที่ ศูนย์เรียนรู้ BlogKub ซึ่งมีบทความสอนทีละฟีเจอร์พร้อมภาพประกอบ

ใครเหมาะกับ BlogKub

ตั้งแต่มือใหม่ที่เพิ่งเริ่มทำบล็อก ไปจนถึงคนที่อยากได้เว็บ Blogger ที่ดูมืออาชีพและทำ SEO ได้จริง โดยไม่ต้องจ้างนักพัฒนา ไม่ว่าจะเป็นบล็อกส่วนตัว บล็อกท่องเที่ยว รีวิวสินค้า คอร์สออนไลน์ หรือเว็บบริษัท BlogKub มีเทมเพลตและฟีเจอร์ให้เลือกใช้อย่างครบถ้วน
